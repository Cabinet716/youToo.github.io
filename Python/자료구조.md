# 자료구조

## list : mutable 한 순서가 있는 객체 집합
```python
# 선언
my_list = [1, 2, 3]
# 추가
my_list.append(4)  # [1, 2, 3, 4]
# 업데이트
my_list[1] = 'two'  # [1, 'two', 3, 4]
# 삭제
del my_list[1]  # [1, 3, 4]
# 초기화
my_list = []
```
