# 생성자
```java
car() {}

car(String model) {
  this(model, "red"); // 같은 클래스 생성자 호출
}

car(String model, String color) {
  this.model = model,
  this.color = color
}
```

# 오버로딩(Overloading)
- 매개변수의 타입과 개수, 순서가 같은 경우가 오버로딩
```java
car(){}
car(String model){}
car(String model, String color){}
//car(String color, String model){} // 오버로딩 아님
car(String model, String color, int maxSpeed){}
```
