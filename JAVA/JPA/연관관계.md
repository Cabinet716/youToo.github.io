# 양방향 관계 

```java
class Team

@OneToMany(mappedBy = "team")
private List<Member> members = new ArrayList<>();
```
### mappedBy 
- 외래키가 있는 곳을 연관관계 주인으로 정하기 ! 
- 양방향 관계인 경우 둘다 값을 넣어주기
```
team.getMember().add(mmber); // 실제 DB insert 안됨. mappedBy는 read 전용, 그러나 1차캐시 됨
```
