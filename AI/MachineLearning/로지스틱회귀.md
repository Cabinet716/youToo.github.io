
# 분류(classification) 은 discrete 한 분류 K개 중 N 개를 선택하는 문제
- Binaray class classification : K = 2, N = 1
- Muilti class classification : K > 2, N = 1
- Multi label classification: K > 2, N > 1
- One class classification: K = 1, N = 1
